# Cafe Management System (C++)

A console-based Cafe Management System developed using C++.

## Features

- Customer ordering system
- Admin login with password
- Receipt generation
- Receipt storage in text file
- View all receipts
- Total sales report
- Multiple item ordering

## Concepts Used

- Functions
- Structures
- Vectors
- File Handling (ifstream / ofstream)
- Loops
- Switch Case
- Conditional Statements

## How to Run

Compile:

```bash
g++ main.cpp -o cafe
```

Run:

```bash
./cafe
```
